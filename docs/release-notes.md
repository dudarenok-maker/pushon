# Release notes

Google Play's **"What's new"** field is capped at **500 characters per
language**. A full app overview belongs in the store listing
([`docs/store-assets/README.md`](store-assets/README.md)); these notes are the
short, per-release summary that fits the limit. Paste the fenced block into the
release's **Release notes** field.

## 1.0.0 — Public Beta

```
PushOn — Public Beta 🎉

Build a push-up habit that sticks — free, private, and fully offline. No account, no ads.

• Set one weekly target; PushOn splits it into daily goals that shift week to week
• Log a set in two taps and watch your daily ring fill
• A streak that pauses (never breaks) on rest days
• Calendar, weekly summary, badges and personal bests
• Gentle reminders, only in your waking hours

This is our first public build — please send bugs and ideas. Thanks for testing!
```
